🍹 Landing Page — Resort AquaViva

Este projeto consiste no desenvolvimento de uma landing page para um resort, criada como prática dos meus estudos em HTML, CSS e JavaScript.

A proposta foi aplicar conceitos de estrutura semântica, organização de layout, responsividade e interação com formulário, criando uma interface moderna e funcional para apresentar o resort e permitir reservas.

🚀 Tecnologias utilizadas

HTML5

CSS3

JavaScript

Flexbox

CSS Grid

Estruturação semântica

🎨 Funcionalidades implementadas

Menu superior fixo com navegação interna

Scroll suave entre seções

Seção Sobre com cards informativos

Galeria de imagens em grid

Seção de serviços do resort

Formulário de reserva

Integração com WhatsApp para envio da reserva

Seção de Perguntas Frequentes (FAQ)

Layout responsivo para diferentes telas

Efeitos hover e animações em elementos

🎯 Objetivo do projeto

Praticar o desenvolvimento de interfaces modernas para web, aplicando conceitos de:

Estrutura semântica em HTML

Estilização com CSS

Layout com Grid e Flexbox

Responsividade com Media Queries

Integração com WhatsApp usando JavaScript

📌 Status

Projeto concluído como parte da minha evolução no desenvolvimento front-end.
No futuro poderá receber melhorias como:

Melhor interação com JavaScript

Sistema de reservas mais avançado

Animações e melhorias visuais
